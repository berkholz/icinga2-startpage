Icingaweb2

Icingaweb2 ist die neue Oberfläche als Ersatz für die Icinga ClassicUI. Hier können die Inforamtionen zu den Hosts und Services eingesehen werden. Zusätzlich lassen sich mit dem Icinga2 Director neue Hosts und Services anlegen.

Grafana

Grafana ist ein Graphingtool zum generieren von Graphen aus verschiedenen Datenquellen. In Grafana kann auf sämtliche Daten aus Icinga2 zurückgegriffen werden.

Strukturdiagramm. Wie sind welche Komponenten miteinander verbunden?

Icinga2 ist eine Weiterentwicklung des Nagios-Forks Icinga 1. Seit der Version 2 hat so einiges Verändert. Icinga2 ist ein sehr Variables automatisierbares Monitoringsystem. Icinga2 ist Modular aufgebaut und kann daher leicht erweitert werden. Grundlegende Funktionen wie Notifications, API, Checker etc. sind ebenfalls als Module realisiert.

Komponenten. Was war jetzt nochmal was?

Die Gesamtstruktur besteht aus den verschiedensten Komponenten, die zum Beispiel Performancedaten graphisch darstellen, Datenbanken bereitstellen etc.

Icinga2

Das Monitoringsystem Icinga2 ist sehr variable und gut automatisierbar. Speziell mit der API-Funktion lassen sich einige Konfigurationen automatisieren.

Icingaweb2

Icingaweb2 ist die moderne Weboberfläche für Icinga2. Ähnlich wie die ClassicUI können darin die Status der Hosts und Services abgerufen werden. Durch Module kann Icingaweb2 erweitert werden, wie zum Beispiel Performancedatenanzeige durch das Graphite-Modul.

IDODB-MySQL

Graphite

Graphite ist für die Speicherung von Performancedaten in einer Datenbank und für die graphische Bereitstellung einer Repräsentation zuständig.

Grafana

Als graphisches Frontend für die Performancedaten kann Grafana genutzt werden. Es bietet dem Benutzer eine graphische Benutzeroberfläche, um Daten aus Quellen jeglicherArt graphisch aufzubereiten. Auch lassen sich verschiedene Daten gegenüberstellen. Damit ist Grafana ein sehr schönes Analysetool für Performancedaten.

MySQL

Als Datenbanksystem für die Performancedaten, die Icingaweb2-Applikation, die Grafana-Applikation und die Graphite-Daten wird ein MySQL-Server eingesetzt.
